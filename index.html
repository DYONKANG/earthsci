파인만 기법 데이터 분석 수업

파인만 기법으로 배우는 고등학교 데이터 분석 (3차시)

수업 목표

  1. 데이터 분석 전 과정을 이해하고, 숨은 의미를 발견하는 즐거움을 느낀다.
  2. 데이터 정제·기초 통계·시각화 개념을 파인만 기법으로 쉽게 설명할 수 있다.
  3. 구글 시트(또는 엑셀) 기능을 활용해 직접 분석하고 결과를 시각화한다.

핵심 교육 방식

  1. 1단계 (선택): 분석할 데이터 주제 선택
  2. 2단계 (설명): 초등학생에게 설명하듯 쉬운 용어와 비유로 개념 설명
  3. 3단계 (확인): 막히는 부분을 찾아 보충 학습
  4. 4단계 (정리): 자신만의 언어로 완벽하게 설명하며 스토리 구성

준비물

  • 교사: PC, 프로젝터, 구글 클래스룸
  • 학생: 1인 1PC, 구글 계정
  • 실습 데이터: Kaggle 전 세계 패스트푸드 영양 정보 CSV
    (필드: Company, Item, Calories, Fat (g), Carbs (g), Protein (g))

수업 목표

  • 데이터 분석이 무엇인지 비유로 이해
  • 행·열 구조 파악, 정렬·필터 기능 실습

핵심 질문

"데이터 분석은 ‘숨은 그림 찾기’ 같은 거야. 어떻게 단서를 잘 찾을까?"

수업 흐름 (50분)

  1. 도입(10분): 질문 유도 → 파인만 기법 소개
  2. 전개(15분): 열·행 설명 → 정렬 시연
  3. 실습(20분): Protein Top5 찾기, 브랜드 필터
  4. 정리(5분): 용어 복습, 다음 차시 예고

수업 목표

  • 결측치·이상치 개념 이해 및 처리
  • 평균·중앙값 등 기초 통계 계산·설명

핵심 질문

"가짜 단서는 어떻게 걸러낼까? → 데이터 정제"

수업 흐름 (50분)

  1. 도입(5분): 지난 시간 복습, 예시 데이터
  2. 전개(15분): 결측치·이상치 파인만 설명, 조건부 서식 시연
  3. 실습(25분): 0 또는 빈칸 행 삭제, AVERAGE/MEDIAN 계산, 브랜드 비교
  4. 정리(5분): 정제 필요성 토의, 다음 차시 예고

수업 목표

  • 막대·파이 차트 선택 및 제작
  • 차트 기반 인사이트 도출 후 한 문장으로 발표

핵심 질문

"결정적 증거(차트)로 이야기하기 → 데이터 시각화"

수업 흐름 (50분)

  1. 도입(5분): 표 vs 차트 비교 → 시각화 중요성
  2. 전개(15분): 막대/파이 차트 파인만 설명, 시연
  3. 실습(25분): 흥미 주제 선정 → 차트 제작 → 결론 작성
  4. 정리(5분): 발표, 토의 → 데이터 분석 마무리
